# Health Service Backend Microservice

### Quickstart

```shell
./start.sh
```
### Usage

* Run units tests
```shell
./gradlew clean test
```

* Build
```shell
./gradlew clean build
```

* Create just database
```shell
docker-compose -f docker-compose-local.yml up
```

### Techs
- Java 8
- Spring Boot Web
- Spring Boot Security
- Spring Boot OAuth2
- Spring Data JPA
- MySQL
- Junit5
- Lombok
- Gradle
- Docker
